# Dilithium SDK Go

Go RPC client for the Dilithium/QSC JSON-RPC and REST surface.

Current scope:

- raw JSON-RPC builders and dispatch
- raw REST helpers
- websocket URL derivation/configuration
- `GetBalance`
- `GetNonce`
- `GetReceipt`
- `GetAddressSummary`
- `GetGasEstimate`
- `GetBaseFee`
- `ResolveName`
- `ReverseResolveName`
- `QueryContract`
- `BuildContractCall`
- `BuildForwarderCall`
- `Simulate`
- `SendCall`
- `WaitForReceipt`
- gas sponsor connector
- messaging connector
