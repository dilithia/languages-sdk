# Dilithia SDK Java

Java-first JVM surface for the Dilithia/QSC RPC line.

Current scope:

- versioned to `0.1.0`
- raw JSON-RPC builders
- websocket URL derivation/configuration
- canonical path/body builders for the core RPC endpoints
- gas estimate and base fee builders
- name service and contract query path builders
- contract call, forwarder, sponsor and messaging builders
- signer adapter interface for higher-level transaction submission
- crypto adapter interface and optional native bridge loader
